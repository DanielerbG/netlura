<h1> 🎬 Netlura</h1>

<h2> 📖 Sobre</h2>
<p>Projeto desenvolvido durante a Imersão Front-End da Alura com IA, com o objetivo de recriar a interface da Netflix utilizando HTML, CSS e JavaScript.

O projeto foi nomeado como *Netlura*, unindo Netflix + Alura 💡</p>

## 🚀 Demonstração

Interface inspirada na Netflix com:

- Seleção de perfis
- Catálogo de filmes e séries
- Carrosséis interativos
- Reprodução automática de trailers ao passar o mouse
- Interface moderna e responsiva

## 🧠 Funcionalidades

### 👤 Seleção de Perfil
- Escolha de usuário ao iniciar
- Armazenamento do perfil no localStorage
- Atualização dinâmica do nome e avatar no catálogo

### 🎞️ Catálogo
- Listagem por categorias:
  - Épicos
  - Séries
  - Para maratonar
- Exibição de informações como:
  - Relevância (%)
  - Classificação indicativa
  - Duração

### ▶️ Reprodução de Trailer
- Vídeos do YouTube iniciam automaticamente ao passar o mouse

### 🎯 Extras
- Barra de progresso em conteúdos assistidos
